## 🚀 Как использовать

1. **Открыть сайт**
   - Откройте `index.html` в браузере
   - Или запустите локальный веб-сервер:
   ```bash
   python -m http.server 8000
   # или
   python3 -m http.server 8000
   ```
   Затем откройте `http://localhost:8000`

2. **Навигация**
   - Используйте навигационную панель для перемещения между страницами
   - Нажимайте на карточки интересов для подробной информации
   - Используйте кнопку "Вернуться" для возврата на предыдущую страницу

3. **Добавление изображений**
   - Поместите изображения в папку `assets/images/`
   - Добавьте ссылки на изображения в HTML файлы

## 📝 Кастомизация

### Изменение цветов
Отредактируйте CSS переменные в `css/style.css`:
```css
:root {
    --primary-blue: #0066cc;
    --dark-blue: #003d7a;
    /* ... другие цвета ... */
}
```

### Изменение контента
Отредактируйте текст в соответствующих HTML файлах в папке `pages/`

### Добавление изображений
1. Поместите изображения в `assets/images/`
2. Добавьте теги `<img>` в HTML:
```html
<img src="../assets/images/your-image.jpg" alt="описание">
```
